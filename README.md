# AngularJS + SystemJS

A Skeleton Project for AngularJS using SystemJS

## Featrues
- ES6 via Babel
- AngularJS with ES6 modules via SystemJS
- SASS

## Setup
```bash
# Step 1
npm install

# Step 2
jspm install
```
## Gulp Tasks
```bash
# Build Production and Serve
gulp serve --production

# Build Development and Serve
gulp serve --development
```
